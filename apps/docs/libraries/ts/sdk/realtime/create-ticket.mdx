---
title: "Create Ticket"
description: "Create a short-lived realtime websocket ticket"
---

<RequestExample>

```ts
const { result, error } = await unprice.realtime.createTicket({
  customerId: "cus_1234567890"
});

if (error) {
  console.error(error.message);
  return;
}

console.log(result);
```

</RequestExample>

<ResponseExample>
```json
{
  "ticket": "rt_...",
  "expiresAt": 1778240000000,
  "projectId": "project_1234567890",
  "customerId": "cus_1234567890"
}
```
</ResponseExample>

## Request

<ParamField body="customerId" type="string" required>
    The customer ID to scope realtime access.
</ParamField>

<ParamField body="projectId" type="string">
    Optional project ID. Only main-project keys can request another project.
</ParamField>
